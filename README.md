# 🇬🇧 English-page — Domina la Gramática Inglesa

Curso completo de gramática inglesa **de principiante a avanzado (A1 → C1)**, con explicaciones claras en español, ejemplos con transcripción fonética (IPA), mapas conceptuales, diagramas y ejercicios de práctica.

> Basado en el libro *"Domina la Gramática Inglesa"* de Jesus Dayan Trigos, ampliado y adaptado a formato interactivo.

## ✨ Características

- 📖 Explicaciones claras en español
- 🔤 Transcripción fonética **IPA** en palabras poco comunes
- 🗺️ **Mapas conceptuales y diagramas** (Mermaid, se renderizan en GitHub)
- 📊 Tablas comparativas y de referencia
- 🏋️ Ejercicios de práctica con respuestas
- 📈 Profundidad progresiva: entre más avanzado el nivel, más detalle

## 🗂️ Estructura del repositorio

```
lessons/
├── A1/  (grammar + vocabulary)   ← básico
├── A2/  (grammar)                ← básico
├── B1/  grammar/  (6 temas)      ← intermedio
├── B2/  grammar/  (5 temas)      ← intermedio alto
├── C1/  grammar/  (5 temas)      ← avanzado
└── EXTRA/ (4 anexos de referencia)
```

## 📝 Cómo estudiar

Sigue el orden de niveles (A1 → C1). Cada lección incluye teoría, ejemplos, diagramas y práctica. Los anexos sirven como consulta rápida en cualquier momento.

🚀 *El aprendizaje de un idioma es constancia. ¡Practica todos los días!*
